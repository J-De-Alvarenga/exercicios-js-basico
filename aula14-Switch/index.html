possuiSwitch

Switch, é uma estrutura de decisão na qual testamos uma expressão e definimos valores possíveis para esta expressão, cada "case" possui os comandos a serem executados, vamos ver a sintaxe do uso do comando.

Switch(teste){
case1: //comandos; break; // break finaliza o case;
case2: //comandos; break;
default: //comandos; // o dafault é última etapa do switch, se nenhum dos cases corresponder ao teste, caímos no default; }
