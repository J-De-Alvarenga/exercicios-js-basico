Estruturas de Decisão

Else If

If, é a principal estrutura de decisão em programação. Ela é definida pelo comando IF, que traduzindo do inglês é SE. Sua sintaxe básica é bem simples.

if(teste){
comandos a serem executados se o teste for verdadeiro
}else{
comandos a serem executados se o teste for falso.}

O else entra em ação se o resultado do teste for falso.

Podemos combinar o if e o else para ter mais opções testadas: Else-if

if(teste){
//Comandos
}else if(teste2){
//Comandos
}else{
//Comandos }
